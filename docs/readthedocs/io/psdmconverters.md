# PSDM Converters
This page lists all converters from and to the PSDM format that are known to us.

## Converters to PSDM

- [simBench2psdm](https://github.com/ie3-institute/simBench2psdm)

The conversion tool is available [here](https://github.com/e2nIEE/simbench), that makes the models available to the [pandapower](https://github.com/e2nIEE/pandapower) simulation software.

SimBench has been research project of Kassel university, Fraunhofer IEE, RWTH Aachen university and TU Dortmund university, aiming at provision of realistic models of electrical distribution grids including time series. The data is availabe [here](https://simbench.de).

- [powerFactory2psdm](https://github.com/ie3-institute/powerFactory2psdm)

This tool is used to convert Power Factory projects into the PSDM format.

- [pp2psdm](https://github.com/ie3-institute/pp2psdm)

Converter for grid models from pandapower into PowerSystemDataModel format.

## Additional Simulation Tools used
- [pandapower](https://github.com/e2nIEE/pandapower)

Pandapower is an easy to use network calculation program aimed to automate the analysis and optimization of power systems.

More information about pandapower can be found [here](https://www.pandapower.org/).
